# Output Workshop DMP Evaluation Metrics

This repository contains draft evaluation metrics and test definitions for assessing reused data information in machine-actionable Data Management Plans (maDMPs). It includes human-readable metric descriptions in Markdown and machine-readable test definitions in JSON-LD.

## Repository structure
- `docs/metrics/` – human-readable metric and test descriptions
- `jsonld/tests/` – machine-readable JSON-LD test definitions
- `examples/` – sample maDMP inputs and expected outputs
